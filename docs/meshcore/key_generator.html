MeshCore Key Generator for NSW

Generate custom Ed25519 keys for MeshCore against the NSW list

About MeshCore Keys

  • Meshcore uses the first two characters of your public key as a node identifier. It is helpful to be able to select your node id to avoid collisions with neighbors if you are building a repeater.
  • This key generator does all processing in your browser - keys never leave your device
